Mahara 포트폴리오 만들기 — 한눈에 보기

계정 → 페이지 생성 → 블록 배치 → 공유·제출 → 점검까지, 실습 흐름대로 설명합니다.

바로 페이지 만들기 공유·제출 보기
시작하기권장 브라우저·파일 정리 페이지 만들기제목·레이아웃·저장 블록 추가텍스트·이미지·파일·임베드 공유·제출권한·기간·미리보기 컬렉션여러 페이지 묶기 체크리스트제출 전 최종 점검

1) 시작하기

필수 준비: 학교에서 제공한 Mahara 접속 주소, 본인 계정(또는 발급 정보), 과제 안내문.

  • 권장 브라우저: 크롬/엣지/사파리 최신 버전
  • 이미지/파일 폴더 정리: images/, docs/ 등
  • 파일명 규칙 예시: 과목_항목_이름_연도.pdf

2) 계정 만들기 & 로그인

  1. 학교가 제공한 Mahara 주소로 접속
  2. 가입 또는 학교 계정으로 로그인(SSO)
  3. 프로필에서 이름·학번(또는 ID)·연락 이메일 확인/수정 → 저장

주의: 비밀번호/복구 이메일을 정확히 등록하세요.

3) 포트폴리오 (페이지) 만들기

  1. 새 페이지 만들기를 클릭하고, 제목을 명확히 입력 (예: “나의 학습 포트폴리오”)
  2. 레이아웃 선택(1~3열 등) → 필요하면 섹션별로 다른 레이아웃 사용
  3. 저장 후 페이지 보기로 미리 확인

제목 팁

과목/기간 포함: “캡스톤 디자인(2025) 포트폴리오”.

섹션 구성

학습 성과 · 성찰 · 증빙자료(문서/영상) · 연락처.

저작권

이미지·도표는 출처를 캡션/본문에 명시.

4) 블록 추가(내용 넣기)

텍스트 블록
  • 문단은 3~5문장으로 간결히, 굵게/리스트로 구조화
  • 성과는 수치/결과 중심(예: “정확도 92%”)
이미지 블록
  • 가로폭 1200px 내외 권장, alt 텍스트/캡션으로 맥락 설명
  • 용량 큰 이미지는 80~85% 품질로 압축
파일/임베드
  • 문서는 PDF로 업로드, 큰 영상은 YouTube(미등록) 링크 임베드
  • 표/데이터는 이미지 대신 문서 링크 제공
테마/레이아웃
  • 테마는 명도 대비가 좋은 기본형 추천
  • 모바일 미리보기로 줄바꿈/겹침 확인

5) 공유·제출

  1. 과목 공지의 제출 방식 확인(비공개 링크 / 그룹 제출 등)
  2. 공유 설정에서 대상(교수자/그룹)과 기간(마감일 포함) 지정
  3. 미리보기로 평가자 관점 확인 → 로그아웃/다른 브라우저로 링크 점검

TIP: 링크가 만료되지 않도록 기간을 과제 마감에 맞추세요.

6) 컬렉션(여러 페이지 묶기)

  1. 새 컬렉션 만들기 → 이름/설명 작성
  2. 포함할 페이지 추가 (순서 변경 가능)
  3. 컬렉션 링크를 제출/공유하면 한 번에 열람

7) 제출 전 체크리스트

FAQ

평가자가 못 본대요

공유 대상·기간을 다시 설정하고 저장하세요. 로그아웃 상태에서 링크 테스트.

용량이 부족해요

영상은 YouTube(미등록)로 임베드, 문서는 PDF 최적화(압축) 후 업로드.

모바일에서 깨져요

이미지 폭 100%, 표는 PDF 링크로 대체. 테마/레이아웃을 단순화.
