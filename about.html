    Mahara 사용 매뉴얼의 전체 흐름은 학생용 가이드에서 확인하세요.

    요약 안내

    흐름 한눈에

    계정 생성 → 페이지 만들기 → 블록 추가 → 공유·제출 → 체크리스트.

    필수 확인

    공유 대상/기간 설정, 로그아웃 상태에서 링크 테스트.

    품질 팁

    성과는 구체적으로, 이미지/표엔 출처 표기, 모바일 미리보기 확인.

    바로가기

    계정 만들기가입/로그인, 프로필 설정 페이지 만들기제목·레이아웃·저장 블록 추가텍스트/이미지/파일/임베드 공유·제출권한·기간·미리보기 컬렉션여러 페이지 묶기 체크리스트제출 전 확인
